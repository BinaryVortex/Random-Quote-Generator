# Random Quote Generator

Random Quote Generator built using HTML, CSS and JavaScript.

![App Screenshot](./Screenshot%202024-07-09%20002942.png)

Overview

A simple, client-side web app that displays a random quote each time you click the button. This project is great for learning DOM manipulation, event handling, and basic styling with CSS.

Tech stack

- HTML
- CSS (41% of the repo)
- JavaScript (35.2% of the repo)

Features

- Show a random quote on button click
- Clean, responsive layout
- Easy to read and extend

Getting started

1. Clone the repository

   git clone https://github.com/BinaryVortex/Random-Quote-Generator.git

2. Open index.html in your browser

   - Double-click `index.html`, or
   - Serve the folder with a simple HTTP server:
     - Python 3: `python -m http.server 8000`
     - Then open `http://localhost:8000` in your browser

How it works

- index.html contains the markup and the button to get a new quote.
- script.js contains the quotes array and the logic to pick and display a random quote.
- style.css contains the styles used to make the app look clean and centered.

Customization

- Add, remove, or edit quotes in `script.js`'s quotes array.
- Change the visual style in `style.css`.
